# SolarLink Social Tent Presentation

Festival booth activation proposal website for the SolarLink Social Tent project.

## Sections

- Brand identity
- Product features
- Embedded cinematic USP demo
- Festival booth concept
- Interactive activation challenge
- ESG impact message

## Run Locally

Open `index.html` directly in a browser, or run the static preview server:

```bash
node scripts/nextent-static-server.cjs
```

Then open:

```text
http://127.0.0.1:8091/
```

## Notes

The demo section embeds the referenced GitHub prototype from `assets/github-demo/` so the presentation can run as a self-contained static site.
